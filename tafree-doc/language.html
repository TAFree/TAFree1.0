TAFree有支援的程式語言

TAFree可以作為批改任何程式作業的平台，只要你知道如何安裝程式語言套件以及撰寫相對應的批改腳本。

如果你是需要負起維護TAFree責任的教職人員，那麼這份文件非常重要。

這份文件和TAFree無關，但是和作業系統相關，這是一份教你如何在Ubuntu 16.04.1 LTS安裝程式語言套件的指南。

換句話說，其他作業系統的使用者必須自行查詢如何在你們的作業系統安裝需要的程式語言套件。

安裝完成後，請務必確認套件版本是否支援作業會用到的功能。


安裝Oracle JDK 9

sudo apt-get install python-software-properties
sudo add-apt-repository ppa:webupd8team/java
sudo apt-get update
sudo apt-get install oracle-java9-installer
sudo java -version

其他JDK


安裝Python 3

sudo apt-get install idle-python3.5
python3 -V

安裝g++ 5

sudo apt-get install g++-5
g++ --version

安裝make

sudo apt-get install make
