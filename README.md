# Build
* Latest Version  
```
sudo git clone -b master http://github.com/TAFree/TAFree1.0.git  
```
* Specified Version  
```
sudo wget http://github.com/tafree/TAFree1.0/archive/2.0.0.tar.gz  
sudo tar zxvf 2.0.0.tar.gz
```
